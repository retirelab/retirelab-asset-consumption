# RetireLab 자산×소비 등급 진단

30대·40대 사용자를 대상으로 **현재 자산등급**, **소비 허용 한도**, **소비등급**, **미래 자산 위치**를 한 화면에서 확인할 수 있도록 만든 단일 HTML 재무 진단 앱입니다.

> 박종경 변호사와 김경필 머니트레이너의 공개 자료를 참고해 RetireLab이 재구성한 비공식 자산·소비 진단 도구입니다.

## 데모

GitHub Pages 배포 후 아래 주소에서 확인할 수 있습니다.

https://retirelab.github.io/retirelab-asset-consumption/

## 주요 기능

- 실제 나이 기준 30대·40대 자산등급 진단
- 자가 보유자와 무주택자의 주거비 구조 분리
- 월 저축률에 따른 미래 순자산 시뮬레이션
- 차량·여행·주거·식비·레저·취미 소비 한도 계산
- 실제 소비 입력 시 소비등급 진단
- 자산등급 × 소비등급 교차 결과 제공
- 결과 문구 복사 및 공유
- 모바일 우선 반응형 디자인
- 서버 전송 없이 브라우저에서만 계산

## 지원 범위

- 지원 나이: 만 30세~49세
- 30대 자산등급 기준
  - S: 순자산 7억 원 이상
  - A: 순자산 5억 원 이상
  - B: 순자산 2억 원 이상
  - C: 순자산 3천만 원 이상
  - D: 순자산 3천만 원 미만
- 40대 자산등급 기준
  - S: 순자산 20억 원 이상
  - A: 순자산 7억 원 이상
  - B: 순자산 3억 원 이상
  - C: 순자산 1억 원 이상
  - D: 순자산 1억 원 미만

## 소비 한도 기본 기준

- 차량: 월소득 6개월치 이내
- 무주택 차량: 월소득 4개월치 이내
- 연간 여행비: 연소득의 5% 이내
- 월 주거비: 월소득의 15% 이내
- 월 식비: 월소득의 25% 이내
- 월 레저·취미비: 월소득의 10% 이내

현재 자산등급에 따라 RetireLab 자체 보정계수가 제한적으로 적용될 수 있습니다.

## 자산 계산 원칙

- 순자산 = 주거자산 + 금융자산 - 부채
- 주택 순자산과 전·월세 보증금에는 투자수익률 0% 적용
- 선택한 투자수익률은 금융 순자산에만 적용
- 주담대 원금상환은 순자산 증가로 인정
- 주담대 이자, 월세, 관리비는 소비성 주거비로 계산

## 파일 구성

```text
retirelab-asset-consumption/
├─ index.html
└─ README.md
```

이 프로젝트는 별도의 빌드 과정이나 외부 라이브러리 없이 `index.html` 하나로 실행됩니다.

## 로컬 실행

1. 저장소를 내려받습니다.
2. `index.html` 파일을 더블 클릭합니다.
3. 웹브라우저에서 바로 실행됩니다.

## GitHub Pages 배포

1. 저장소를 Public으로 설정합니다.
2. GitHub 저장소에서 `Settings`로 이동합니다.
3. `Pages`를 선택합니다.
4. Source를 `Deploy from a branch`로 선택합니다.
5. Branch는 `main`, 폴더는 `/(root)`로 선택합니다.
6. 저장 후 배포 완료를 기다립니다.

## 개인정보 보호

입력한 소득·자산·소비 정보는 서버로 전송되지 않습니다. 모든 계산은 사용자의 브라우저 안에서만 처리됩니다.

## 주의사항

이 앱은 개인 참고용 단순 시뮬레이션입니다.

- 공식 재무상담, 세무상담 또는 투자 권유가 아닙니다.
- 미래 수익률, 세금, 물가, 소득 변화, 시장 손실은 실제와 다를 수 있습니다.
- 자산등급과 소비 기준은 공개 자료를 바탕으로 RetireLab이 재구성한 비공식 기준입니다.
- 중요한 재무 의사결정은 개인 상황에 맞춰 별도로 검토해야 합니다.
